# Shadow Arena: Ninja Backgammon

## Overview
Shadow Arena is a competitive, tournament-grade Backgammon game featuring a dark ninja dojo theme. It is being built on the Injective blockchain for real-money wagering. The gameplay is strictly deterministic, prioritizing strategic fairness and psychology over randomness manipulation.

## Tech Stack
- Frontend: React 18, TypeScript, TailwindCSS, React Router
- Blockchain: Injective (CosmWasm smart contract) - Planned
- Assets/Styling: Custom visual assets (Backgammon board, Doubling Cube, Character Avatars) with heavy use of CSS blend modes, drop-shadows, and dark themes.

## Key Principles
- Tournament-grade fairness (No mechanic alters dice probability).
- Tension driven by the doubling cube and player decisions.
- Dark, cinematic, competitive atmosphere.
- All dramatic effects must be toggleable (Fast Mode).
